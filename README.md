# Advent of code 2023 solutions by Lauri Gustafsson

Solutions are written for Zig 0.12.0-dev.1830. To build a solution run:
```
zig build-exe dayN.zig
```

And run with:
```
./dayN < input
```
Or from clipboard (in wlroots-based wayland compositors)
```
wl-paste | ./dayN
```
